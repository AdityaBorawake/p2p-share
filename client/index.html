Frequently asked questions

How do I send a large file?

Click "Create Room", share the link with the other person, wait for them to join, then select your file and send. No upload needed — the file goes directly to their device.

Is there a file size limit?

No. Because files travel directly between devices (peer-to-peer), there is no server size restriction. You can send files of any size.

Are my files stored on a server?

Never. Files are never uploaded to any server. The only thing our server does is help the two devices find each other. After that, the connection is direct.

Does it work on mobile?

Yes. BigFileSendP2P works on any modern browser — iPhone, Android, Mac, Windows, Linux. No app install required.

What is peer-to-peer (P2P) file sharing?

P2P means files go directly from one device to another, without passing through a central server. This makes transfers faster, more private, and removes any file size limits.
