Frequently asked questions

How do I send a large file for free?

Click "Create Room", share the link with the other person, wait for them to join, then select your file and send. No upload needed — the file goes directly to their device at full speed.

Is there a file size limit?

No. Because files travel directly between devices (peer-to-peer), there is no server size restriction. You can send files of any size — from small documents to 100GB+ video files.

Are my files stored on a server?

Never. Files are never uploaded to any server. The only thing our server does is help the two devices find each other. After that, the connection and the file transfer are completely direct.

Does it work on mobile phones?

Yes. BigFileSendP2P works on any modern browser — iPhone, Android, Mac, Windows, Linux. No app install required. Just open the link and go.

What is peer-to-peer (P2P) file sharing?

P2P means files go directly from one device to another, without passing through a central server. This makes transfers faster, completely private, and removes file size limits — because there's no server bottleneck.

How is this different from WeTransfer or Google Drive?

WeTransfer and Google Drive upload your file to their servers first, then the recipient downloads it — with size limits and storage costs. BigFileSendP2P skips the server entirely. The file goes directly between devices, so there's no size limit, no wait, and no one else ever sees your file.

How fast is the transfer?

As fast as your network allows. On the same WiFi network, transfers typically run at 5–50 MB/s. Over the internet it depends on both users' connection speeds, but there's no artificial throttling — you always get the maximum your connection can deliver.

Can I send multiple files at once?

Yes. You can select multiple files or drag and drop a batch — they all appear in the send queue and transfer one after another automatically.
